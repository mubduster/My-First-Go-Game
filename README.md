# My first go game
## What it is?
  A basic game made in go so i can learn game development without the need of any fancy game eniges like Unity and Unreal Engine, which all although are good, but it's better to know to code a game than just designing it in my opinion. (add further documentation)

## How to play?

  - W A S D to move
  - thats it.
